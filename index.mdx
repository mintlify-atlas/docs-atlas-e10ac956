## Overview

Vantix is a RESTful sales management system designed with Clean Architecture principles, providing a robust foundation for building scalable sales applications. Built on ASP.NET Core 10 and Entity Framework Core, it offers a well-structured approach to managing customer data and sales operations.

## Architecture Layers

Vantix follows Clean Architecture principles with clearly separated concerns:

<Steps>
  <Step title="Domain Layer">
    Contains entity models and core business logic, independent of infrastructure
  </Step>
  <Step title="Application Layer">
    Defines application-specific business rules and use cases
  </Step>
  <Step title="Infrastructure Layer">
    Implements data access with Entity Framework and external service integrations
  </Step>
  <Step title="API Layer">
    Exposes HTTP endpoints using ASP.NET Core Web API
  </Step>
</Steps>
